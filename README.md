## üí° Overview & Mission

This repository is a public log of my **#100DaysOfCode** challenge, focusing exclusively on **Data Structures and Algorithms (DSA)** implemented in **Java**. My mission is to:

* **Deepen Foundational Knowledge:** Build a rock-solid understanding of core DSA principles.
* **Master Problem Solving:** Develop robust algorithmic thinking and efficient coding solutions.
* **Create Educational Content:** Provide clear, concise, and well-explained notes and solutions to help other learners.
* **Showcase My Skills:** Build a visible portfolio for **Software Development Internship** opportunities.
* **Engage with the Community:** Connect with fellow developers, seek feedback, and contribute to collaborative learning.

This repo covers everything from basic Java constructs to complex algorithmic paradigms, including detailed **LeetCode Problem Solutions** with explanations and dry runs.

## üìÇ Repository Structure: Your Learning Path

This repository is organized to guide you through a structured learning journey.

* **`Day_XX/` Folders:** Each folder represents a day's dedicated effort on a specific topic. Inside, you'll find:
    * `notes.md`: **Comprehensive explanations**, analogies, and theoretical insights for the day's concept.
    * `Practice_Problems/`: Foundational **Java code solutions** for the day's topic.
    * `Modified_Code/` (where applicable): Code adapted for **visualization tools** (e.g., PythonTutor) to aid understanding.

* **`LeetCode_Solutions/` (New!):** A dedicated, growing collection of **LeetCode problem solutions**, organized by DSA topic. Each problem entry includes:
    * **Optimized Java Code:** Clean and efficient implementations.
    * **Detailed `Explanation.md`:** Step-by-step logic, approach, and **Time/Space Complexity analysis**.
    * **`Dry_Run.md` (Optional):** Visual tracing of execution for complex algorithms, helping you understand the flow.

## üöÄ Getting Started with This Repo

Want to explore the code or learn alongside?

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/verma-suraj/Java-DSA-100Days-Challenge.git](https://github.com/verma-suraj/Java-DSA-100Days-Challenge.git)
    ```
2.  **Navigate to any specific day's folder or a LeetCode solution:**
    ```bash
    cd Java-DSA-100Days-Challenge/Day_XX
    # OR
    cd Java-DSA-100Days-Challenge/LeetCode_Solutions/Arrays/Problem_001_TwoSum
    ```
3.  **Explore the `notes.md` file** for detailed explanations of concepts.
4.  **Run Java code:** Compile and execute `.java` files from your IDE or terminal.
    ```bash
    javac YourProblemName.java
    java YourProblemName
    ```
